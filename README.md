# Java Basic Applications ☕

This repository contains simple yet functional Java console applications to practice core programming concepts.

## 📁 Projects Included:

### 🏦 ATM Simulator (`ATM.java`)
* Features: Password authentication, balance checking, deposits, and withdrawals.
* Concepts: `do-while` loops, `if-else` logic, and formatted output.

### 🛒 Supermarket Billing System (`Supermarket_Billing_System.java`)
* Features: Calculating totals for multiple products, applying a 10% discount for orders over 500 DH, and adding TVA.
* Concepts: User input handling, arithmetic calculations, and conditional discounting.
